# Prompt Craft

The **Prompt Craft** feature introduces a new, interactive prompt builder that allows users to design, test, and refine prompts directly within the application. Key capabilities include:

- **Visual prompt editor** with syntax highlighting and real‑time validation.
- **Prompt templates** that can be cloned, modified, and shared.
- **Live preview** of prompt execution results against a sandboxed model.
- **Versioning** and rollback for prompt iterations.

> **Getting started**
> 1. Navigate to the *Prompt Craft* tab.
> 2. Click **Create New Prompt**.
> 3. Use the editor to compose your prompt and click **Run** to see results.

For detailed usage instructions, see the [Prompt Craft Guide](docs/prompt-craft.md).
